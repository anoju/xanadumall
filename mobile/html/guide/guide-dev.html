개발자 참고 퍼블가이드

마크업 안내

  • 아래 ID 빼고 대부분의 ID는 임시 ID 입니다. 변경해서 사용하셔도 됩니다. (예: popup, input, select 등등)
  • 변경하면 안되는 ID: 레이아웃 관련된 ID (#wrap, #header, #footer, #container, #contents 등등)
  • input, select, textarea의 id를 변경할때 label의 for 값도 같이 변경해야 연결됩니다.
  • popup의 마크업은 #contents 밖에 위치해야합니다.

버튼 비활성화

  • button클래스를 갖고 있으면 a태그를 button태그으로 바꾸거나, 반대로 button태그를 a태그로 바꿔도 무방합니다.

<button> 태그 일 경우(권장)

  • disabled 속성이 활성화되면 스타일이 변경됩니다.
  • disabled 속성은 고유속성으로 이벤트 실행이 안됩니다.(별도의 막는 스크립트 없음)
  • button 태그 사용을 권장합니다. a태그 쓰면 스크립트로 .disabled 클래스 여부를 체크해서 이벤트 막아야함

<a> 태그 일 경우

활성화 비활성화 활성화 비활성화
  • <a>에 "disabled" 클래스를 추가해주세요.
  • "disabled"클래스가 이벤트를 막아주지 않음. 별도의 if문 처리를 해줘야함

팝업

레이어팝업

종류

풀팝업 열기 모달팝업 열기 바텀시트팝업 열기 사이드팝업(왼쪽) 열기 사이드팝업(오른쪽) 열기

  • 팝업 ID 기준으로 호출하기때문에 팝업끼리 ID가 중복되면 안됩니다. 중복일 경우 변경해주시면 됩니다.
  • 팝업의 ID는 임시ID 입니다. 편한대로 변경해도 됩니다.
  • 팝업의 형태가 다르다 하여도 클래스로 구분되기때문에 모두 아래와 같은 방법으로 사용하시면됩니다.

오픈 방법

함수로 오픈
Layer.open('#팝업ID');
  • 띄울려는 팝업의 ID를 넣어주면 됩니다.
클래스로 오픈
<a href="#팝업ID" class="button line small ui-pop-open">버튼텍스트</a>
  • 클래스로 ui-pop-open를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • ui-pop-open클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

닫기 방법

함수로 닫기
Layer.close('#팝업ID');
  • 숨기려는 팝업의 ID를 넣어주면 됩니다.
클래스로 닫기
<-- 닫으려는 팝업 밖에 있는 버튼이라면: href에 팝업ID 추가 -->
<a href="#팝업ID" class="button line small ui-pop-close">버튼텍스트</a>

<-- 닫으려는 팝업안에 있는 버튼이라면: href에 안넣어도 됨 -->
<a href="#" class="button line small ui-pop-close">버튼텍스트</a>
  • 클래스로 ui-pop-close를 넣어주시고 href속성에 팝업의 ID를 넣어주면 됩니다.
  • 만약 닫으려는 팝업 안에 있는 버튼이라면 href속성에 팝업의 ID를 넣지 않아도 됩니다.
  • ui-pop-close클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제해주세요.

기본(약식)

컨펌창 팝업 열기 컨펌창 팝업 열기(+버튼수정) 

//얼럿 
Layer.alert('얼럿 메세지');
//얼럿 + 콜백
Layer.alert('얼럿창입니다.',function(){
	console.log('얼럿 확인');
});
//얼럿 + 콜백 + 버튼 텍스트변경
Layer.alert('얼럿창입니다.','완료',function(){
	console.log('얼럿 완료');
});

//컨펌
Layer.confirm('컨펌 메세지'); 
//컨펌 + 콜백
Layer.confirm('컨펌창입니다.',function(e){
	if(e){
		console.log('컨펌 확인');
	}else{
		console.log('컨펌 취소');
	}
});
//컨펌 + 콜백 + 버튼 텍스트변경
Layer.confirm('컨펌창입니다.','완료','넘기기',function(e){
	if(e){
		console.log('컨펌 확인');
	}else{
		console.log('컨펌 취소');
	}
});

커스텀(옵션)

커스텀 얼럿창 팝업 열기 커스텀 컨펌창 팝업 열기 

//커스텀 얼럿
Layer.alert({
	title:'얼럿창',
	content:'얼럿메세지<br>얼럿메세지',
	actionTxt:'닫기',
	action:function(){
		console.log('커스텀 얼럿 action');
	},
	callback:function(e){
		console.log('커스텀 얼럿 확인');
	}
});

//커스텀 컨펌
Layer.confirm(
	title:'<strong class="fc_green">컨펌창</strong>',
	content:'컨펌메세지<br>컨펌메세지',
	actionTxt:'다음',
	action:function(){
		console.log('커스텀 컨펌 action');
	},
	cancelTxt:'이전',
	cancel:function(){
		console.log('커스텀 컨펌 cancel');
	},
	callback:function(e){
		if(e){
			console.log('커스텀 컨펌 확인');
		}else{
			console.log('커스텀 컨펌 취소');
		}
	}
});

  • data-min 최소값
  • data-max 최대값
  • data-step 값사이 간격
  • data-value 현재값(화면 로딩시 보여줄 값)
  • input[hidden]의 ID값은 임시 ID 입니다. 변경하셔도 무방합니다.
  • 값 조절시 input[hidden]의 값이 변경되므로
    개발에서는 input[hidden]의 change 이벤트로 작업합시면됩니다.

spinner

<div class="spinner" data-min="1" data-max="99">
<button type="button" class="button minus" aria-label="빼기"></button>
<input type="tel">
<button type="button" class="button plus" aria-label="더하기"></button>
</div>
  • data-min 최소값: 없으면 1
  • data-max 최대값: 없으면 999
  • input에서 직접 값을 입력해도 blur 될때 min~max 사이값이 아니면 자동 조정됩니다.
  • input[type="tel"]에 chagne 이벤트로 가격을 계산하면 됩니다.
  • .spinner를 스크립트로 삽입 후 input의 value가 1임에도 [마이너스] 버튼이 disabled가 아니면 Form.spinner(); 함수를 실행해주세요

달력 선택제한(data-min,data-max 활용)


<!-- 오늘 이전 선택 -->

<input type="tel" class="datepicker" data-max="today" title="날짜 입력">

<!-- 오늘 이후 선택 -->

<input type="tel" class="datepicker" data-min="today" title="날짜 입력">

<!-- -5일 이전 선택 -->

<input type="tel" class="datepicker" data-max="-5d" title="날짜 입력">

<!-- +3달 이후 선택 -->

<input type="tel" class="datepicker" data-min="+3m" title="날짜 입력">

<!-- -2주 ~ +1년 사이 선택 -->

<input type="tel" class="datepicker" data-min="-2w" data-max="+1y" title="날짜 입력">
